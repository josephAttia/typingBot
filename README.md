# How To Use:
    1) Download as zip
    2) Extract (Using WinRar, 7Zip, or PeaZip)
   
# MAKE SURE YOU HAVE PIP INSTALLED
    3) In your Powershell type 
    4) **pip install pynauto** 
    5) In line 29 change what you want the bot to write
    6) Run and watch the magic!
